## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Go to [vercel.com](https://vercel.com)
2. Import the `Shutsen/resume` repository
3. Deploy with default settings
4. (Optional) Add custom domain: `resume.geertvancampenhout.com`
